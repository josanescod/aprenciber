# Aprenciber
A web platform for hands-on cybersecurity labs in isolated environments

The project is composed of two parts:
- **Backend**: Python application using FastAPI
- **Frontend**: Vue application + Tailwind CSS

## Requirements

- Python 3.13.5
- fnm 1.38.1
- node v24.13.1
- pnpm 10.33.0
- Docker 29.4.1
- ttyd 1.7.7
- PostgreSQL 16
